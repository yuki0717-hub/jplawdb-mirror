別表の作成順序とデータフロー
決算書→各別表→申告書の全体像

■全体フロー
決算書（財務諸表）
  ・PL: 当期純利益（税引後）
  ・BS: 資産・負債・純資産
    │
    ↓ PL当期純利益を転記
STEP 1: 税務調整別表（別表六～十六）を先に作成
  ・B16(二) → 定額法の減価償却超過額を算出
  ・B16(六) → 繰延資産の償却超過額を算出
  ・B16(八) → 一括償却資産の損金算入限度超過額を算出
  ・B16(九) → 特別償却準備金の積立超過額を算出
  ・B15 → 交際費の損金不算入額を算出
  ・B14(二) → 寄附金の損金不算入額を算出
  ・B13(一) → 国庫補助金等の圧縮記帳額を算出
  ・B11(一の二)(二) → 貸倒引当金・返品調整引当金の繰入超過額を算出
  ・B10(五) → 特許権等の譲渡等の特別控除額を算出
  ・B08(一) → 受取配当の益金不算入額を算出
  ・B08(二) → 外国子会社配当の益金不算入額を算出
  ・B07(一) → 繰越欠損金の控除額を算出
  ・B06(一) → 所得税額控除の額を算出
  ・B06(二) → 外国税額控除の額を算出
  ・B06(二の二)(三)(四)(四の二)(五) → 外国税額控除の明細・繰越を算出
  ・B06(五の二) → 分配時調整外国税相当額を算出
  ・B06(六) → 特別控除額（税額控除総括）を算出
  ・B06(二十四) → 賃上げ税額控除の額を算出
  ・B17(一) → 過少資本税制の損金不算入額を算出
  ・B17(二)(二の二)(二の三) → 過大支払利子税制の損金不算入額を算出
  ・B17(三)～(三の九) → CFC合算課税の課税対象金額を算出
  ・B17(四) → 移転価格税制の国外関連者情報を整理
    │
    ↓ 各別表の計算結果を転記
STEP 2: 別表四（所得の金額の計算）
  当期純利益（PL）
  ＋加算（損金不算入: 交際費,寄附金,償却超過 等）
  －減算（益金不算入: 受取配当,欠損金控除 等）
  ＝ 所得金額（又は欠損金額）
    │
    ↓ 所得金額を転記
STEP 3: 別表一（確定申告書）
  所得金額 × 税率（23.2%、中小15%/800万以下）
  ＋ 留保金課税額（B03(一)から・特定同族会社のみ）
  － 税額控除（B06系の控除額）
  － 中間申告額
  ＝ 差引確定法人税額
  ＋ 地方法人税（法人税額×10.3%）
    │
    ↓ 法人税額を転記
STEP 4: 別表五(二)（租税公課の納付状況）
  期首未納税額 → 当期発生 → 当期納付 → 期末未納
  ・法人税額（B01から）
  ・住民税額（第六号様式から）
  ・事業税額（第六号様式から）
    │
    ↓ 期末未納税額等を転記
STEP 5: 別表五(一)（利益積立金額の計算）
  各項目: 期首残高 ＋ 増加 － 減少 ＝ 期末残高
  ・未納法人税（← B05(二)の期末未納）
  ・繰越損益金（← 決算書の繰越利益剰余金）
  ・各留保項目（← B04の留保②欄）
  ★ 別表四と検算必須 → /check/kenzan.txt

STEP 6: 別表二（同族会社判定）
  ※ 他の別表と独立して作成可能

■作成順序の理由
なぜB06～B16が先か？
  B04は各税務調整の「集約先」。交際費(B15)、寄附金(B14)、
  減価償却(B16)等の計算結果がないとB04を完成できない。
  必ず個別の税務調整別表を先に作成する。

なぜB05(一)が最後か？
  B05(一)はB04の「留保②」欄の金額を転記する。
  B04が確定しないと作成できない。またB05(二)の未納税額も必要。
  よってB05(一)は最後になる。

B17(三の三)部分課税対象金額 → B04加算(留保)（CFC部分合算・非金融）
B17(三の四)金融子会社等部分課税対象金額 → B04加算(留保)（CFC部分合算・金融）
B17(三の五)控除対象外国法人税額 → B06(二)（CFC外国税額控除）
B17(三の七)益金不算入額 → B04減算(社外流出)（CFC課税済配当）
B01確定法人税額 → B05(二)
B05(二)期末未納 → B05(一)未納法人税
B04留保②各行 → B05(一)増③各行

■地方税との関係
法人税の申告と同時に地方税（住民税・事業税）の申告も行う。
・B04所得金額 → 第六号様式（事業税の課税標準）
・B01法人税額 → 第六号様式（住民税の法人税割の課税標準）
・第六号様式の事業税額 → B05(二)の事業税行
法人税と地方税は一体で作業するのが実務の常。
